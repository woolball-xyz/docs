# Getting Started with Wool Ball

<Note>
Follow these steps to begin using Wool Ball's distributed browser-based processing network.
</Note>

## Setup Steps

<Steps>
  <Step title="Create an Account">
    Start by [creating a free account](https://woolball.xyz/Identity/Account/Manage/) and receive $1 credit to test our services.
  </Step>
  <Step title="Get Your API Key">
    After registration, obtain your API key from the dashboard. This key will be used to authenticate your requests.

    ```bash
    # Example API key format
    WOOL_BALL_API_KEY=wb_xxxxxxxxxxxxxxxxxxxx
    ```
  </Step>
  <Step title="Make Your First API Call">
    Here's a simple example using our Text-to-Speech API:

    ```javascript
    async function generateSpeech() {
      const response = await fetch('https://api.woolball.xyz/v1/text-to-speech', {
        method: 'POST',
        headers: {
          'Authorization': 'Bearer YOUR_API_KEY',
          'Content-Type': 'application/json',
        },
        body: JSON.stringify({
          text: 'Hello from Wool Ball!',
          voice: 'en-US-female-1'
        })
      });
      
      const audio = await response.blob();
      return audio;
    }
    ```
  </Step>
</Steps>
