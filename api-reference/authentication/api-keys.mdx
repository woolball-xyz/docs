# API Keys

API keys are used to authenticate your requests to the Wool Ball API. You can manage your API keys through the [Wool Ball Dashboard](https://woolball.xyz/Identity/Account/Manage/).

## Getting Your API Key

1. Log in to your [Wool Ball Dashboard](https://woolball.xyz/Identity/Account/Manage/)
2. Navigate to the API Keys section
3. Click "Generate New API Key"
4. Copy and securely store your API key

## API Key Types

- **Development Keys**: Use these for testing and development
- **Production Keys**: Use these for live applications

## Rate Limits

API requests are subject to rate limiting based on your subscription plan. The current limits are:

- Free tier: 100 requests per day
- Pro tier: 10,000 requests per day
- Enterprise tier: Custom limits

Contact support for more information about rate limits and enterprise plans.
