---
title: 'Translation'
api: 'POST /v1/translation'
description: 'Translate text between languages'
---

Translate text from one language to another.

### Body

<ParamField body="text" type="string" required>
  Text to translate
</ParamField>

<ParamField body="targetLanguage" type="string" required>
  Target language code
</ParamField>

<ParamField body="sourceLanguage" type="string">
  Source language code (optional, will be auto-detected if not provided)
</ParamField>

### Response

<ResponseField name="data" type="string">
  The translated text
</ResponseField>

### Status Codes

<ResponseField name="200" type="object">
  OK - Successful request
</ResponseField>

<ResponseField name="400" type="object">
  Bad Request - Validation error occurred
</ResponseField>

<ResponseField name="401" type="object">
  Unauthorized - Authentication failed
</ResponseField>

<ResponseField name="402" type="object">
  Payment Required
</ResponseField>

```bash
curl -X POST https://api.woolball.xyz/v1/translation \
  -H "Authorization: Bearer <YOUR_API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Hello, world!",
    "targetLanguage": "pt",
    "sourceLanguage": "en"
  }'
```
