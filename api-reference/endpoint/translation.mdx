Translate text between different languages using our distributed browser network.

## Request

<ParamField body="Text" type="string" required>
  The text to translate
</ParamField>

<ParamField body="SrcLang" type="string" required>
  Source language code (e.g., "por_Latn" for Portuguese)
</ParamField>

<ParamField body="TgtLang" type="string" required>
  Target language code (e.g., "eng_Latn" for English)
</ParamField>

<RequestExample>
```bash
curl -X POST https://api.woolball.xyz/v1/translation \
-H "Authorization: Bearer API_KEY" \
-H "Content-Type: application/json" \
-d '{"Text": "Olá, mundo!", "SrcLang": "por_Latn", "TgtLang": "eng_Latn"}'
```

```javascript
const apiBaseUrl = "https://api.woolball.xyz";
const endpoint = "/v1/translation";

fetch(apiBaseUrl + endpoint, {
  method: 'POST',
  headers: {
      'Authorization': 'Bearer API_KEY',
      'Content-Type': 'application/json'
  },
  body: JSON.stringify({
      Text: "Olá, mundo!",
      SrcLang: "por_Latn",
      TgtLang: "eng_Latn"
  })
})
.then(res => res.json())
.then(data => {
  console.log("Translated Text:", data.data);
})
.catch(console.error);
```

```python
import requests

api_base_url = "https://api.woolball.xyz"
endpoint = "/v1/translation"

headers = {
  "Authorization": "Bearer API_KEY",
  "Content-Type": "application/json"
}
payload = {
  "Text": "Olá, mundo!",
  "SrcLang": "por_Latn",
  "TgtLang": "eng_Latn"
}

response = requests.post(api_base_url + endpoint, headers=headers, json=payload)
data = response.json()
print(f"Translated Text: {data['data']}")
```
</RequestExample>
