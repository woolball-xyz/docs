---
title: 'Translation'
api: 'POST /v1/translation'
description: 'Translate text between languages'
---

Translate text from one language to another using advanced translation models.

### Body

<ParamField body="text" type="string" required>
  Text to translate
</ParamField>

<ParamField body="srcLang" type="string" required>
  Source language code (e.g., "en" for English)
</ParamField>

<ParamField body="tgtLang" type="string" required>
  Target language code (e.g., "es" for Spanish)
</ParamField>

### Response

<ResponseField name="data" type="object">
  The translation result
</ResponseField>

<RequestExample>
```bash
curl -X POST https://api.woolball.xyz/v1/translation \
  -H "Authorization: Bearer <YOUR_API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Hello, how are you?",
    "srcLang": "en",
    "tgtLang": "es"
  }'
```
</RequestExample>

<ResponseExample>
```json
{
  "data": {
    "translation": "¡Hola, ¿cómo estás?"
  }
}
```
</ResponseExample>
