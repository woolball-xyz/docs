Translate text between languages using our distributed browser network.

## Request

<ParamField body="text" type="string" required>
  The text to translate
</ParamField>

<ParamField body="source_language" type="string">
  Source language code (e.g., "pt", "en", "es"). Default is auto-detect.
</ParamField>

<ParamField body="target_language" type="string" required>
  Target language code (e.g., "pt", "en", "es")
</ParamField>

<RequestExample>
```bash
curl -X POST https://api.woolball.xyz/v1/translation \
  -H "Authorization: Bearer API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Hello World",
    "source_language": "en",
    "target_language": "pt"
  }'
```

```javascript
fetch('https://api.woolball.xyz/v1/translation', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    text: 'Hello World',
    source_language: 'en',
    target_language: 'pt'
  })
})
.then(res => res.json())
.then(data => {
  console.log('Translated Text:', data.data);
})
.catch(console.error);
```

```python
import requests

response = requests.post(
    'https://api.woolball.xyz/v1/translation',
    headers={
        'Authorization': 'Bearer API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'text': 'Hello World',
        'source_language': 'en',
        'target_language': 'pt'
    }
)
data = response.json()
print(f"Translated Text: {data['data']}")
```
</RequestExample>
