---
title: 'Image Classification'
api: 'POST /v1/image-classification'
description: 'Classify images into predefined categories'
---

Classify images into predefined categories using computer vision models.

### Body

<ParamField body="image" type="file" required>
  Image file to classify
</ParamField>

### Response

<ResponseField name="data" type="object">
  The image classification results
</ResponseField>

<RequestExample>
```bash
curl -X POST https://api.woolball.xyz/v1/image-classification \
  -H "Authorization: Bearer <YOUR_API_KEY>" \
  -F "image=@image.jpg"
```
</RequestExample>

<ResponseExample>
```json
{
  "data": {
    "classifications": [
      {
        "label": "cat",
        "confidence": 0.98
      }
    ]
  }
}
```
</ResponseExample>
