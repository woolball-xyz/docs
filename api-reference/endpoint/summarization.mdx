---
title: 'Summarization'
api: 'POST /v1/summarization'
description: 'Generate summaries of text content'
---

Generate concise summaries of longer text content.

### Body

<ParamField body="text" type="string" required>
  Text content to summarize
</ParamField>

<ParamField body="maxLength" type="integer">
  Maximum length of the generated summary
</ParamField>

### Response

<ResponseField name="data" type="object">
  The generated summary
</ResponseField>

<RequestExample>
```bash
curl -X POST https://api.woolball.xyz/v1/summarization \
  -H "Authorization: Bearer <YOUR_API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Long text content to summarize...",
    "maxLength": 100
  }'
```
</RequestExample>

<ResponseExample>
```json
{
  "data": {
    "summary": "Concise summary of the input text"
  }
}
```
</ResponseExample>
