Generate concise summaries of text content using our distributed browser network.

## Request

<ParamField body="text" type="string" required>
  The text to summarize
</ParamField>

<ParamField body="max_length" type="integer">
  Maximum length of the summary in characters. Default is 150
</ParamField>

<RequestExample>
```bash
curl -X POST https://api.woolball.xyz/v1/summarization \
  -H "Authorization: Bearer API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Long text content to summarize...",
    "max_length": 150
  }'
```

```javascript
fetch('https://api.woolball.xyz/v1/summarization', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    text: 'Long text content to summarize...',
    max_length: 150
  })
})
.then(res => res.json())
.then(data => {
  console.log('Summary:', data.data);
})
.catch(console.error);
```

```python
import requests

response = requests.post(
    'https://api.woolball.xyz/v1/summarization',
    headers={
        'Authorization': 'Bearer API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'text': 'Long text content to summarize...',
        'max_length': 150
    }
)
data = response.json()
print(f"Summary: {data['data']}")
```
</RequestExample>
