Generate concise summaries from text using our distributed browser network.

## Request

<ParamField body="text" type="string" required>
  The text to summarize
</ParamField>

<RequestExample>
```bash
curl -X POST https://api.woolball.xyz/v1/summarization \
-H "Authorization: Bearer API_KEY" \
-H "Content-Type: application/json" \
-d '{
  "text": "Artificial intelligence (AI) is intelligence demonstrated by machines, as opposed to the natural intelligence displayed by animals including humans."
}'
```

```javascript
const fetch = require('node-fetch');

const apiBaseUrl = "https://api.woolball.xyz";
const endpoint = "/v1/summarization";

const body = {
  text: "Artificial intelligence (AI) is intelligence demonstrated by machines, as opposed to the natural intelligence displayed by animals including humans."
};

fetch(apiBaseUrl + endpoint, {
  method: 'POST',
  headers: {
      'Authorization': 'Bearer API_KEY',
      'Content-Type': 'application/json'
  },
  body: JSON.stringify(body)
})
.then(res => res.json())
.then(data => {
  console.log("Summary:", data.data);
})
.catch(console.error);
```

```python
import requests

api_base_url = "https://api.woolball.xyz"
endpoint = "/v1/summarization"

headers = {
  "Authorization": "Bearer API_KEY",
  "Content-Type": "application/json"
}

data = {
  "text": "Artificial intelligence (AI) is intelligence demonstrated by machines, as opposed to the natural intelligence displayed by animals including humans."
}

response = requests.post(api_base_url + endpoint, headers=headers, json=data)
data = response.json()
print(f"Summary: {data['data']}")
```
