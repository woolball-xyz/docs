---
title: 'Image Zero Shot Classification'
api: 'POST /v1/image-zero-shot'
description: 'Classify images using zero-shot learning with custom labels'
---

Classify images into custom categories using zero-shot learning, allowing for flexible image classification without pre-training.

### Body

<ParamField body="image" type="file" required>
  Image to classify
</ParamField>

<ParamField body="labels" type="string" required>
  JSON array of candidate labels for classification. Example: ["dog", "cat", "bird", "fish"]
</ParamField>

### Response

<ResponseField name="data" type="object">
  The classification results with confidence scores for each label
</ResponseField>

<RequestExample>
```bash
curl -X POST https://api.woolball.xyz/v1/image-zero-shot \
  -H "Authorization: Bearer <YOUR_API_KEY>" \
  -F "image=@image.jpg" \
  -F 'labels=["dog", "cat", "bird", "fish"]'
```
</RequestExample>

<ResponseExample>
```json
{
  "data": {
    "classifications": [
      {
        "label": "dog",
        "confidence": 0.95
      },
      {
        "label": "cat",
        "confidence": 0.03
      },
      {
        "label": "bird",
        "confidence": 0.01
      },
      {
        "label": "fish",
        "confidence": 0.01
      }
    ]
  }
}
```
</ResponseExample>
