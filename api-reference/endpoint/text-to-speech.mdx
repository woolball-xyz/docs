---
title: 'Text to Speech'
api: 'GET /v1/text-to-speech/{targetLanguage}'
description: 'Convert text to speech in different languages'
---

Convert text to speech in various languages using advanced text-to-speech models.

### Path Parameters

<ParamField path="targetLanguage" type="string" required>
  Target language for speech synthesis. Available options: "en" (English), "pt" (Portuguese), "es" (Spanish)
</ParamField>

### Query Parameters

<ParamField query="text" type="string" required>
  The text to convert to speech
</ParamField>

### Response

<ResponseField name="data" type="string">
  Base64 encoded audio data
</ResponseField>

<RequestExample>
```bash
curl -X GET "https://api.woolball.xyz/v1/text-to-speech/en?text=Hello%20World" \
  -H "Authorization: Bearer <YOUR_API_KEY>"
```
</RequestExample>

<ResponseExample>
```json
{
  "data": "base64_encoded_audio_data"
}
```
</ResponseExample>
