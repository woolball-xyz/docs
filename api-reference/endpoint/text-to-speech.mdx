Convert text to speech using our distributed browser network.

## Request

<ParamField query="text" type="string" required>
  The text to convert to speech
</ParamField>

<ParamField path="targetLanguage" type="string" required>
  Target language code (e.g., "pt", "en", "es")
</ParamField>

<RequestExample>
```bash
curl -s -H "Authorization: Bearer API_KEY" "https://api.woolball.xyz/v1/text-to-speech/pt?text=Olá%20mundo"
```

```javascript
const input = "Olá mundo";
const targetLanguage = "pt"; // Alterar para "en", "es" conforme necessário
var url = `https://api.woolball.xyz/v1/text-to-speech/${targetLanguage}?text=${input}`;

fetch(url, {
  headers: { Authorization: "Bearer API_KEY" }
})
.then(res => res.json())
.then(data => {
  const audioBase64 = data.data;
  const byteLength = atob(audioBase64).length;
  const sizeInMB = (byteLength / (1024 * 1024)).toFixed(2);

  console.log(`O tamanho do áudio é de ${sizeInMB} MB`);
})
.catch(console.error);
```

```python
import requests
import base64

api_base_url = "https://api.woolball.xyz"
target_language = "pt"  # Alterar para "en", "es" conforme necessário
endpoint = f"/v1/text-to-speech/{target_language}?text=Olá mundo"

headers = {"Authorization": "Bearer API_KEY"}
response = requests.get(api_base_url + endpoint, headers=headers)
data = response.json()
audio_base64 = data["data"]
audio_bytes = base64.b64decode(audio_base64)

print(f"Tamanho do áudio: {len(audio_bytes) / (1024 * 1024):.2f} MB")
```
</RequestExample>
