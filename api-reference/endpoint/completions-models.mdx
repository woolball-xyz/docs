---
title: 'Completion Models'
api: 'GET /v1/models/completions'
description: 'Get available models for text completion'
---

Returns a list of available models for text completion tasks.

### Response

<ResponseField name="data" type="array">
  List of available completion models
  <Expandable title="Model">
    <ResponseField name="model" type="string">
      The model identifier
    </ResponseField>
    <ResponseField name="dtype" type="string">
      The data type configuration
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash
curl -X GET https://api.woolball.xyz/v1/models/completions \
  -H "Authorization: Bearer <YOUR_API_KEY>"
```
</RequestExample>

<ResponseExample>
```json
{
  "data": [
    {
      "model": "gpt2",
      "dtype": "float16"
    }
  ]
}
```
</ResponseExample>
