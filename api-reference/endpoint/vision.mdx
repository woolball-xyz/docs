---

Analyze and describe image content using our distributed browser network.

## Request

<ParamField body="image" type="blob" required>
  The image file to analyze
</ParamField>

<ParamField body="prompt" type="string" required>
  The prompt to guide the image analysis (e.g., "Describe the content of the image")
</ParamField>

<RequestExample>
```bash
curl -X POST https://api.woolball.xyz/v1/vision \
-H "Authorization: Bearer API_KEY" \
-F "image=@img.png" \
-F "prompt=Describe the content of the image"
```

```javascript
const fs = require('fs');
const FormData = require('form-data');
const fetch = require('node-fetch');

const apiBaseUrl = "https://api.woolball.xyz";
const endpoint = "/v1/vision";

const formData = new FormData();
formData.append('image', fs.createReadStream('img.png'));
formData.append('prompt', 'Describe the content of the image');

fetch(apiBaseUrl + endpoint, {
  method: 'POST',
  headers: {
      'Authorization': 'Bearer API_KEY'
  },
  body: formData
})
.then(res => res.json())
.then(data => {
  console.log("Analysis Result:", data.data);
})
.catch(console.error);
```

```python
import requests

api_base_url = "https://api.woolball.xyz"
endpoint = "/v1/vision"

headers = {"Authorization": "Bearer API_KEY"}
files = {'image': open('img.png', 'rb')}
data = {'prompt': 'Describe the content of the image'}

response = requests.post(api_base_url + endpoint, headers=headers, files=files, data=data)
data = response.json()
print(f"Analysis Result: {data['data']}")
```
</RequestExample>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="string">
  The generated description or analysis of the image
</ResponseField>
