---
title: 'Vision'
api: 'POST /v1/vision'
description: 'General purpose computer vision endpoint'
---

General purpose computer vision endpoint for analyzing images.

### Body

<ParamField body="image" type="file" required>
  Image to analyze
</ParamField>

### Response

<ResponseField name="data" type="object">
  The vision analysis results
</ResponseField>

<RequestExample>
```bash
curl -X POST https://api.woolball.xyz/v1/vision \
  -H "Authorization: Bearer <YOUR_API_KEY>" \
  -F "image=@image.jpg"
```
</RequestExample>

<ResponseExample>
```json
{
  "data": {
    "analysis": {
      "description": "A detailed description of the image content",
      "tags": ["detected", "objects", "and", "concepts"],
      "confidence": 0.95
    }
  }
}
```
</ResponseExample>
