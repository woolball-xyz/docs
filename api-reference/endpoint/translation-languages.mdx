---
title: 'Translation Languages'
api: 'GET /v1/languages'
description: 'Get available languages for translation'
---

Returns a list of available languages for translation.

### Response

<ResponseField name="data" type="array">
  List of available languages
  <Expandable title="Language">
    <ResponseField name="code" type="string">
      The language code
    </ResponseField>
    <ResponseField name="name" type="string">
      The language name
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash
curl -X GET https://api.woolball.xyz/v1/languages \
  -H "Authorization: Bearer <YOUR_API_KEY>"
```
</RequestExample>

<ResponseExample>
```json
[
  {
    "code": "eng_Latn",
    "name": "English"
  },
  {
    "code": "por_Latn",
    "name": "Portuguese"
  }
]
```
</ResponseExample>
