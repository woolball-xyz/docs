---
title: 'Translation Languages'
api: 'GET /v1/languages'
description: 'Get available languages for translation'
---

Returns a list of available languages for translation services.

### Response

<ResponseField name="data" type="array">
  List of available languages
  <Expandable title="Language">
    <ResponseField name="code" type="string">
      The language code (e.g., "en", "es", "pt")
    </ResponseField>
    <ResponseField name="name" type="string">
      The language name (e.g., "English", "Spanish", "Portuguese")
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash
curl -X GET https://api.woolball.xyz/v1/languages \
  -H "Authorization: Bearer <YOUR_API_KEY>"
```
</RequestExample>

<ResponseExample>
```json
{
  "data": [
    {
      "code": "en",
      "name": "English"
    },
    {
      "code": "es",
      "name": "Spanish"
    },
    {
      "code": "pt",
      "name": "Portuguese"
    }
  ]
}
```
</ResponseExample>
