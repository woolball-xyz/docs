The Translation Languages endpoint provides a list of all supported languages for translation services using Flores-200 language codes.

## Response

<ResponseField name="data" type="array">
  Array of supported language objects
</ResponseField>

### Language Object

<ResponseField name="code" type="string">
  Flores-200 language code, which includes both language and script information (e.g., "eng_Latn" for English in Latin script)
</ResponseField>

<ResponseField name="name" type="string">
  Language name in English, including script information when relevant
</ResponseField>

## Understanding Language Codes

The language codes follow the Flores-200 format: `[language]_[script]`

Common scripts include:
- `Latn`: Latin script
- `Hans`: Simplified Chinese characters
- `Hant`: Traditional Chinese characters
- `Cyrl`: Cyrillic script
- `Arab`: Arabic script
- `Deva`: Devanagari script
- `Jpan`: Japanese writing system
- `Hang`: Korean Hangul
- `Thai`: Thai script
- `Hebr`: Hebrew script
