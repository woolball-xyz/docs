Classify images into custom categories without predefined training using our distributed browser network.

## Request

<ParamField body="image" type="blob" required>
  The image file to classify
</ParamField>

<ParamField body="labels" type="array" required>
  Array of possible labels for classification
</ParamField>

<ParamField body="multi_label" type="boolean">
  Allow multiple labels per image. Default is false
</ParamField>

<RequestExample>
```bash
curl -X POST https://api.woolball.xyz/v1/zero-shot-image-classification \
  -H "Authorization: Bearer API_KEY" \
  -H "Content-Type: multipart/form-data" \
  -F "image=@image.jpg" \
  -F "labels=[\"sunny\", \"cloudy\", \"rainy\"]" \
  -F "multi_label=false"
```

```javascript
const formData = new FormData();
formData.append('image', imageBlob);
formData.append('labels', JSON.stringify(['sunny', 'cloudy', 'rainy']));
formData.append('multi_label', false);

fetch('https://api.woolball.xyz/v1/zero-shot-image-classification', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer API_KEY'
  },
  body: formData
})
.then(res => res.json())
.then(data => {
  console.log('Classifications:', data.data);
})
.catch(console.error);
```

```python
import requests

files = {
    'image': open('image.jpg', 'rb')
}
data = {
    'labels': ['sunny', 'cloudy', 'rainy'],
    'multi_label': False
}

response = requests.post(
    'https://api.woolball.xyz/v1/zero-shot-image-classification',
    headers={
        'Authorization': 'Bearer API_KEY'
    },
    files=files,
    data=data
)
data = response.json()
print(f"Classifications: {data['data']}")
```
</RequestExample>
