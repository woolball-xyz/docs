Classify images into custom categories without predefined training using our distributed browser network.

## Request

<ParamField body="image" type="blob" required>
  The image file to classify
</ParamField>

<ParamField body="labels" type="array" required>
  Array of possible labels for classification (e.g., ["cat", "dog", "tree", "human"])
</ParamField>

<RequestExample>
```bash
curl -X POST https://api.woolball.xyz/v1/image-zero-shot \
-H "Authorization: Bearer API_KEY" \
-F "image=@img.png" \
-F "labels=[\"cat\", \"dog\", \"tree\", \"human\"]"
```

```javascript
const fs = require('fs');
const FormData = require('form-data');
const fetch = require('node-fetch');

const apiBaseUrl = "https://api.woolball.xyz";
const endpoint = "/v1/image-zero-shot";

const formData = new FormData();
formData.append('image', fs.createReadStream('img.png'));
formData.append('labels', JSON.stringify(["cat", "dog", "tree", "human"]));

fetch(apiBaseUrl + endpoint, {
  method: 'POST',
  headers: {
      'Authorization': 'Bearer API_KEY'
  },
  body: formData
})
.then(res => res.json())
.then(data => {
  data.data.forEach(result => {
      console.log(result);
  });
})
.catch(console.error);
```

```python
import requests

api_base_url = "https://api.woolball.xyz"
endpoint = "/v1/image-zero-shot"

headers = {"Authorization": "Bearer API_KEY"}
files = {'image': open('img.png', 'rb')}
data = {'labels': '["cat", "dog", "tree", "human"]'}

response = requests.post(api_base_url + endpoint, headers=headers, files=files, data=data)
data = response.json()
for result in data['data']:
  print(f"Label: {result['label']}, Score: {result['score']}")
```
</RequestExample>
