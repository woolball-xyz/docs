---
title: 'Speech to Text'
api: 'POST /v1/speech-to-text'
description: 'Convert audio to text using speech recognition'
---

Convert audio files to text using advanced speech recognition models.

### Body

<ParamField body="audio" type="file" required>
  Audio file to transcribe
</ParamField>

### Query Parameters

<ParamField query="model" type="string">
  Speech recognition model to use. Default: "onnx-community/whisper-base"
</ParamField>

<ParamField query="outputLanguage" type="string">
  Target language for the transcription output. Example: "pt"
</ParamField>

<ParamField query="returnTimestamps" type="boolean">
  Whether to return timestamps for each transcribed segment
</ParamField>

### Response

<ResponseField name="data" type="object">
  The transcription result
</ResponseField>

<RequestExample>
```bash
curl -X POST https://api.woolball.xyz/v1/speech-to-text \
  -H "Authorization: Bearer <YOUR_API_KEY>" \
  -F "audio=@audio.mp3" \
  -G \
  -d "model=onnx-community/whisper-base" \
  -d "outputLanguage=pt" \
  -d "returnTimestamps=true"
```
</RequestExample>

<ResponseExample>
```json
{
  "data": {
    "text": "Transcribed text content",
    "segments": [
      {
        "text": "Segment text",
        "start": 0.0,
        "end": 2.5
      }
    ]
  }
}
```
</ResponseExample>
