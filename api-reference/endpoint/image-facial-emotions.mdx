---
title: 'Image Facial Emotions'
api: 'POST /v1/image-facial-emotions'
description: 'Detect facial emotions in images'
---

Analyze images to detect and classify facial emotions.

### Body

<ParamField body="image" type="file" required>
  Image file to analyze for facial emotions
</ParamField>

### Response

<ResponseField name="data" type="object">
  The facial emotion analysis results
</ResponseField>

<RequestExample>
```bash
curl -X POST https://api.woolball.xyz/v1/image-facial-emotions \
  -H "Authorization: Bearer <YOUR_API_KEY>" \
  -F "image=@face.jpg"
```
</RequestExample>

<ResponseExample>
```json
{
  "data": {
    "emotions": [
      {
        "emotion": "happy",
        "confidence": 0.95
      }
    ]
  }
}
```
</ResponseExample>
