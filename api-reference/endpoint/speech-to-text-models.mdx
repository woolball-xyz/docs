---
title: 'Speech to Text Models'
api: 'GET /v1/models/speech-to-text'
description: 'Get available speech-to-text models'
---

Returns a list of available models for speech recognition.

### Response

<ResponseField name="data" type="array">
  List of available speech recognition models
  <Expandable title="Model">
    <ResponseField name="id" type="string">
      The unique identifier of the model
    </ResponseField>
    <ResponseField name="name" type="string">
      The name of the model
    </ResponseField>
    <ResponseField name="description" type="string">
      Description of the model's capabilities
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash
curl -X GET https://api.woolball.xyz/v1/models/speech-to-text \
  -H "Authorization: Bearer <YOUR_API_KEY>"
```
</RequestExample>

<ResponseExample>
```json
{
  "data": [
    {
      "id": "whisper-base",
      "name": "Whisper Base",
      "description": "OpenAI's Whisper base model for speech recognition"
    }
  ]
}
```
</ResponseExample>
