Generate images from characters or emojis using our distributed browser network.

## Request

<ParamField body="character" type="string" required>
  The character or emoji to convert to an image
</ParamField>

<ParamField body="style" type="string">
  Style of the generated image (e.g., 'cartoon', 'realistic', 'artistic'). Default is 'cartoon'
</ParamField>

<ParamField body="size" type="string">
  Size of the output image ('small', 'medium', 'large'). Default is 'medium'
</ParamField>

<RequestExample>
```bash
curl -X POST https://api.woolball.xyz/v1/character-to-image \
  -H "Authorization: Bearer API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "character": "üê∂",
    "style": "cartoon",
    "size": "medium"
  }'
```

```javascript
fetch('https://api.woolball.xyz/v1/character-to-image', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    character: 'üê∂',
    style: 'cartoon',
    size: 'medium'
  })
})
.then(res => res.json())
.then(data => {
  console.log('Generated Image:', data.data);
})
.catch(console.error);
```

```python
import requests

response = requests.post(
    'https://api.woolball.xyz/v1/character-to-image',
    headers={
        'Authorization': 'Bearer API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'character': 'üê∂',
        'style': 'cartoon',
        'size': 'medium'
    }
)
data = response.json()
print(f"Generated Image: {data['data']}")
```
</RequestExample>
