---
title: 'Zero Shot Classification'
api: 'POST /v1/zero-shot-classification'
description: 'Classify text without training using zero-shot learning'
---

Classify text into categories without pre-training using zero-shot learning models.

### Body

<ParamField body="text" type="string" required>
  Text to classify
</ParamField>

<ParamField body="labels" type="array" required>
  Array of possible classification labels
</ParamField>

### Response

<ResponseField name="data" type="object">
  The classification results
</ResponseField>

<RequestExample>
```bash
curl -X POST https://api.woolball.xyz/v1/zero-shot-classification \
  -H "Authorization: Bearer <YOUR_API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "I love this product!",
    "labels": ["positive", "negative", "neutral"]
  }'
```
</RequestExample>

<ResponseExample>
```json
{
  "data": {
    "labels": [
      {
        "label": "positive",
        "score": 0.95
      },
      {
        "label": "neutral",
        "score": 0.03
      },
      {
        "label": "negative",
        "score": 0.02
      }
    ]
  }
}
```
</ResponseExample>
