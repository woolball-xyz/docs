Classify text into custom categories without predefined training using our distributed browser network.

## Request

<ParamField body="text" type="string" required>
  The text to classify
</ParamField>

<ParamField body="labels" type="array" required>
  Array of possible labels for classification
</ParamField>

<ParamField body="multi_label" type="boolean">
  Allow multiple labels per text. Default is false
</ParamField>

<RequestExample>
```bash
curl -X POST https://api.woolball.xyz/v1/zero-shot-classification \
  -H "Authorization: Bearer API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "The weather is sunny and warm",
    "labels": ["sunny", "cloudy", "rainy"],
    "multi_label": false
  }'
```

```javascript
fetch('https://api.woolball.xyz/v1/zero-shot-classification', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    text: 'The weather is sunny and warm',
    labels: ['sunny', 'cloudy', 'rainy'],
    multi_label: false
  })
})
.then(res => res.json())
.then(data => {
  console.log('Classifications:', data.data);
})
.catch(console.error);
```

```python
import requests

response = requests.post(
    'https://api.woolball.xyz/v1/zero-shot-classification',
    headers={
        'Authorization': 'Bearer API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'text': 'The weather is sunny and warm',
        'labels': ['sunny', 'cloudy', 'rainy'],
        'multi_label': False
    }
)
data = response.json()
print(f"Classifications: {data['data']}")
```
</RequestExample>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="array">

    <ResponseField name="label" type="string">

    <ResponseField name="confidence" type="number">
