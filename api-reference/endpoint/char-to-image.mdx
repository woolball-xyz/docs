---
title: 'Character to Image'
api: 'GET /v1/char-to-image'
description: 'Convert characters to images'
---

Convert text characters into corresponding images.

### Query Parameters

<ParamField query="char" type="string" required>
  Character to convert to image
</ParamField>

### Response

<ResponseField name="data" type="string">
  Base64 encoded image data
</ResponseField>

<RequestExample>
```bash
curl -X GET https://api.woolball.xyz/v1/char-to-image \
  -H "Authorization: Bearer <YOUR_API_KEY>" \
  -G \
  -d "char=A"
```
</RequestExample>

<ResponseExample>
```json
{
  "data": "base64_encoded_image_data"
}
```
</ResponseExample>
