---
title: 'Character to Image'
api: 'GET https://api.woolball.xyz/v1/char-to-image'
description: 'Convert a character into an image'
---

Convert a single character into an image using our distributed browser network.

## Request

<ParamField query="character" type="string" required>
  The character to convert into an image (e.g., "A")
</ParamField>

<RequestExample>
```bash
curl -X GET "https://api.woolball.xyz/v1/char-to-image?character=A" \
-H "Authorization: Bearer API_KEY"
```

```javascript
const character = "A";
const apiBaseUrl = "https://api.woolball.xyz";
const endpoint = "/v1/char-to-image";

fetch(`${apiBaseUrl}${endpoint}?character=${character}`, {
  headers: { 'Authorization': 'Bearer API_KEY' }
})
.then(res => res.json())
.then(data => {
  console.log("Image Base64:", data.data);
})
.catch(console.error);
```

```python
import requests

api_base_url = "https://api.woolball.xyz"
endpoint = "/v1/char-to-image"
character = "A"

headers = {"Authorization": "Bearer API_KEY"}
response = requests.get(f"{api_base_url}{endpoint}?character={character}", headers=headers)
data = response.json()
print(f"Image Base64: {data['data']}")
```
</RequestExample>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="string">
  Base64 encoded image data
</ResponseField>

<ResponseExample>
```json
{
  "success": true,
  "data": "iVBORw0KGgoAAAANSUhEUgAA..."  // Base64 encoded image
}
```
</ResponseExample>
